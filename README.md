# Support Ticket Classification & Prioritization

An ML system that automatically classifies customer support tickets
into categories and assigns priority levels (High/Medium/Low).

## Tools Used
- Python, Jupyter Notebook
- scikit-learn, NLTK, pandas

## Dataset
Kaggle - Customer Support Ticket Dataset
https://www.kaggle.com/datasets/suraj520/customer-support-ticket-dataset

## Models
- Logistic Regression for Ticket Category classification
- Logistic Regression for Priority Level prediction
